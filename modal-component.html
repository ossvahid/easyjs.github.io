Modal Component


What IS Modal Component ?

The component modal is a powerful tool for creating custom modals. You can easily activate it with a few lines of code and create professional modals effortlessly. Building a modal normally comes with complexities, but with the Easy JS Modal component, you can easily create any modal.

note : In the examples below, I used the W3CSS library to style the modals. You can use any other method to design

# Basic Example

Step One : make modal Body with HTML 

<div class="mymodal w3-white w3-round-medium w3-center w3-xlarge w3-panel w3-padding-24" style="display: none;">
    This IS A Basic Modal
</div>
<button class="mymodal-handler w3-button w3-medium w3-padding-small w3-black w3-round-medium">
     open modal
</button>

Step Two : call Easy js Modal Component 

import { EasyJS } from '/components/dist/easyjs.js'; 
EasyJS.component('modal', {
      wrapperClass: 'mymodal',
      handlerClass: 'mymodal-handler',
});

# Advanced Example

let make a modal with advanced options

Step One : make modal Body with HTML 

<div class="mymodal2 w3-display-container w3-white w3-round-medium w3-center w3-panel w3-padding-24"
style="max-width: fit-content; display: none;">
<button
class="closemodal w3-btn w3-pale-red w3-text-red w3-hover-red w3-round-large w3-cell w3-display-topright w3-margin">
X
</button>
<h3>
This IS A Advanced Modal
</h3>
<p class="w3-large">
In this example we have used more advanced settings
</p>
<button class="acceptModal w3-btn w3-pale-blue w3-text-blue w3-hover-blue w3-round-large w3-cell w3-margin-top">
Accept
</button>
</div>

Step Two : call Easy js Modal Component 

import { EasyJS } from '/components/dist/easyjs.js'; 
EasyJS.component('modal', {
    wrapperClass: 'mymodal2',
    handlerClass: 'mymodal-handler2',
    closeButtonClass: 'closemodal',
    acceptButtonClass: 'acceptModal',
    position: 'top-center',
    animationIn: 'fadeInDown',
    animationOut: 'fadeOutUp',
    animationSpeed: '0.7s',
    onAccept: () => {
        alert('accepted')
      }
});

Finnaly :

Modal Component Options

on below part you can see total Modal Component Options

@ Required Options

  • wrapperClass : string () => modal wrapper class

@ Optional Options

  • handlerClass : string () => modal handler class
  • animationIn : string (bounceIn) => modal animation in
  • animationOut : string (bounceOut) => modal animation out
  • animationSpeed : string (1s) => modal animation speed
  • closeByClickModal : bool (true) => close modal when user clicked back modal
  • onOpen : callback() => on open modal callback
  • onClose : callback() => on close modal callback
  • onAccept : callback() => on accept modal callback
  • position : string (center) => modal position (center|top-center|top-left|top-right|bottom-center|bottom-left|bottom-right)
  • acceptButtonClass : string () => accept button class
  • closeButtonClass : string () => close button class
  • padding : string (20px) => modal padding default is 20px
  • backgroundColor : string (rgba(0, 0, 0, 0.85)) => modal background Color
