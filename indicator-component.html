Indicator Component Options

on below part you can see total Indicator Component Options

@ Required Options

  • wrapperClass : string () => indicator wrapper class
  • itemsClass : string () => indicator items class
  • indicatorClass : string () => indicator class

@ Optional Options

  • mode : string (both) => indicator move ratio ( both | x | y )
  • indicatorMovingSpeed : string (1s) => indicator move speed
  • indicatorMove : callback(indicator,targetItem) => this callback function fired when indicator moving
  • indicatorStop : callback(indicator,targetItem) => this callback function fired when indicator stop
  • events : array(['mousemove', 'click']) => indicator events : any js event is accepted
  • indicatorPositionMode : string (absolute) => indicator position mode ( fixed | relative | static | absolute )
  • indicatorRatio : string (center | end) => indicator Ratio
  • indicatorYmargin : string (0px) => indicator vertical margin
  • indicatorXmargin : string (0px) => indicator horizontal margin
  • cover : bool (true) => if set true , indicator will cover with current target size
  • eventsMaxSensitivity : bool (false) => this is event sensitivity mode , if set true, any event on target elm with have max sensitivity
  • indicatorDefaultHidden : bool (false) => indicator will default hide, and when client start working with registered events indicator will showing and when client stop working with events indicator will hidden again
  • indicatorAnimationIn : string () => add an animation for indicator when client work with events
  • indicatorAnimationOut : string () => add an animation for indicator when client stop working with events
  • indicatorAnimationSpeed : string (1s) => indicator animation speed
