What Is Accordion Compoment ?

The Accordion component is a useful and simple tool for creating Accordion lists anywhere

note : In the examples below, I used the W3CSS library to style the modals. You can use any other method to design

Step Two : call accordion component 

import { EasyJS } from './easyjs/components/dist/easyjs.js';
EasyJS.component('accordion',{
    'wrapperClass': 'accordion-wrapper',
    'itemClass': 'item',
    'itemDataClass': 'data',
    'itemHeadingClass': 'heading',
    'icon':'plus',
    'defaultItemOpen': 1,
    'allItemOpenable': true,
    'itemActiveClass': 'active-item'
})

accordion Component Options

on below part you can see total Accordion Component Options

@ Required Options

  • wrapperClass : string () => accordion wrapper class
  • itemClass : string () => accordion items class
  • itemDataClass : string () => accordion items data wrapper class
  • itemHeadingClass : string () => accordion Heading class

@ Optional Options

  • allItemOpenable : bool (true) => if set true all item can expanded if set false only one item can expanded
  • defaultItemOpen : int () => ...
  • itemActiveClass : string () => add a class for expanded item
  • icon : string (arrow) => item icon (arrow | plus)
