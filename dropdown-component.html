Dropdown Component


What IS Dropdown Component ?

You can create a dynamic box with the Dropdown component

Example One (make a basic dropdown)

first create dropdown html body and dropdown handler 

 <button class="w3-button w3-teal w3-round-medium w3-padding-small dropdown-handler">
                    See Dropdown
                </button>

<div class="dropdown-wrapper w3-card w3-padding w3-white w3-round-large ">
                    <h4>
                        A Simple DropDown
                    </h4>
                    <p>
                        Lorem ipsum, dolor sit amet consectetur adipisicing elit. Consectetur labore.
           </p>
</div>

Then Call dropdown component 

import { EasyJS } from './easyjs/components/dist/easyjs.js'
    EasyJS.component('dropdown', {
        wrapperClass: 'dropdown-wrapper',
        handlerClass: 'dropdown-handler',
        width: '300px',
        events: ['click'],
})

you can add new animation 

import { EasyJS } from './easyjs/components/dist/easyjs.js'
    EasyJS.component('dropdown', {
        wrapperClass: 'dropdown-wrapper',
        handlerClass: 'dropdown-handler',
        width: '300px',
        events: ['click'],
        animationIn: 'bounceIn',
        animationOut: 'bounceOut',
        animationSpeed: '1s',
})
